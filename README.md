  <h2 style="color: #1a202c; margin-top: 0; font-size: 2.5em; font-weight: 800; border-bottom: 3px solid #4A90E2; padding-bottom: 15px;">1. Overview & Philosophy</h2>
  <p style="color: #333; line-height: 1.8; font-size: 1.05em;">
    <strong>Ethereal Dashboard</strong> is an ultra-premium, high-performance, privacy-first browser homepage engine designed to execute entirely at the client browser layer. The architecture operates with zero external dependencies, delivering a complete sandboxed computing experience.
  </p>
  <ul style="color: #333; line-height: 2; font-size: 1.05em;">
    <li><strong style="color: #4A90E2;">0 External Server Connections:</strong> All computing overhead and asset loading are sandboxed locally.</li>
    <li><strong style="color: #4A90E2;">0 Telemetry / Data Tracking:</strong> Guarantees absolute user privacy with no underlying metrics phone-home code.</li>
    <li><strong style="color: #4A90E2;">0 External API Key Dependencies:</strong> Eliminates operational reliance on paid, rate-limited, or volatile third-party cloud engines.</li>
  </ul>
  <p style="color: #555; line-height: 1.8; font-size: 1.05em; font-style: italic; background-color: rgba(74, 144, 226, 0.05); padding: 15px; border-radius: 8px;">
    Everything executes locally using client compute power, enabling fully deterministic behavior, ultra-low latency canvas rendering, and absolute user data sovereignty.
  </p>
</div>

  <h2 style="color: #1a202c; border-bottom: 3px solid #4A90E2; padding-bottom: 15px; font-size: 2.5em; font-weight: 800; margin-top: 0;">2. Technical Capabilities & Core Features</h2>
  
  <h3 style="color: #4A90E2; font-size: 1.8em; font-weight: 700; margin-top: 25px;">⚙️ Spatial Layout & Canvas Engine</h3>
  <ul style="line-height: 2; font-size: 1.05em; color: #333;">
    <li><strong>Premium Glassmorphic UI:</strong> Features fluid, draggable widget orchestration for advanced spatial desktop personalization.</li>
    <li><strong>Fluid Animation Physics:</strong> Utilizes Framer Motion-driven interaction mechanics for natural structural UI transitions.</li>
    <li><strong>GPU-Accelerated Styling:</strong> Implements granular background media filters (blur, contrast, brightness) rendered via hardware-accelerated CSS pipelines to eliminate runtime repaints.</li>
  </ul>

  <h3 style="color: #4A90E2; font-size: 1.8em; font-weight: 700; margin-top: 25px;">🧠 Client-Side Machine Learning Inference</h3>
  <ul style="line-height: 2; font-size: 1.05em; color: #333;">
    <li><strong>Local Foreground Isolation:</strong> Implements fully offline AI subject isolation and background-removal workflows for uploaded custom imagery.</li>
    <li><strong>WASM Neural Network Pipeline:</strong> Executes local WebAssembly neural network inference optimized to remain secure and compliant within strict Chrome Extension Manifest V3 Contexts.</li>
  </ul>

  <h3 style="color: #4A90E2; font-size: 1.8em; font-weight: 700; margin-top: 25px;">🔍 Spotlight Search Terminal</h3>
  <ul style="line-height: 2; font-size: 1.05em; color: #333;">
    <li><strong>Programmable Slash Commands:</strong> Intercepts and routes custom terminal triggers (such as <code>/yt</code>, <code>/gh</code>, and <code>/g</code>) straight from the interactive search interface.</li>
    <li><strong>Dynamic Asset Ingestion:</strong> Features a processing string injection pipeline that dynamically renders raw local Base64 shortcut icons on active command match detection.</li>
    <li><strong>Safe Dispatch Redirection:</strong> Implements secure query-injection handling for clean, controlled target search engine dispatches.</li>
  </ul>

  <h3 style="color: #4A90E2; font-size: 1.8em; font-weight: 700; margin-top: 25px;">📌 DOM Hijacking Metadata Engine</h3>
  <ul style="line-height: 2; font-size: 1.05em; color: #333;">
    <li><strong>Tab Layer Overrides:</strong> Houses a local document-title override engine to bypass standard browser tab text restrictions.</li>
    <li><strong>Canvas-to-Favicon Streaming:</strong> Handles real-time rendering of raw HTML5 canvas Base64 image streams directly into the physical browser tab favicon.</li>
    <li><strong>Zero Network Cost:</strong> Executes visual tab identity updates instantly with zero network roundtrips or static hosting requirements.</li>
  </ul>

  <h4 style="color: #1a202c; margin-top: 25px; font-size: 1.4em; font-weight: 700; border-left: 4px solid #4A90E2; padding-left: 15px;">Architectural Optimization Drivers:</h4>

  <h2 style="color: #1a202c; font-size: 2.8em; font-weight: 900; border-bottom: 4px solid #4A90E2; padding-bottom: 15px; margin-top: 0;">5. Deployment & Installation Guide</h2>

  <h3 style="color: #4A90E2; font-size: 2em; font-weight: 800; margin-top: 30px;">📂 Initial Project Download</h3>
  <p style="line-height: 1.8; font-size: 1.05em; color: #333;">Before proceeding to browser installation, pull the portable project files onto your local machine:</p>
  <ol style="line-height: 2; font-size: 1.05em; color: #333;">
    <li>Click the green <strong>Code</strong> button at the top right of this repository.</li>
    <li>Select <strong>Download ZIP</strong> from the dropdown menu.</li>
    <li>Extract the downloaded <code style="background-color: #f0f0f0; padding: 3px 8px; border-radius: 4px;">.zip</code> file archive to your preferred local storage directory.</li>
  </ol>

  <h3 style="color: #4A90E2; font-size: 2em; font-weight: 800; margin-top: 30px;">🛠️ Automated Setup <em style="font-size: 0.9em;">(Chromium Browsers: Chrome, Brave, Edge, Opera)</em></h3>
  <p style="line-height: 1.8; font-size: 1.05em; color: #333;">The repository contains an automated orchestration pipeline designed to handle system environment setup and asset bundling automatically.</p>
  <ol style="line-height: 2; font-size: 1.05em; color: #333;">
    <li>Open your extracted project directory and navigate into the root <code style="background-color: #f0f0f0; padding: 3px 8px; border-radius: 4px;">browser_exten</code> folder.</li>
    <li>Locate the file named <code style="background-color: #f0f0f0; padding: 3px 8px; border-radius: 4px;">install_and_launch.bat</code>.</li>
    <li>Right-click on the script and select <strong>"Run as Administrator"</strong>.</li>
    <li>The script will automatically verify dependencies, parse project files, and execute a production compile using the following routines:
<pre style="background-color: #2b2b2b; color: #f8f8f2; padding: 15px; border-radius: 8px; overflow-x: auto; font-size: 1.05em;"><code>pnpm install
pnpm build</code></pre>
    </li>
    <li>On completion, it will natively inject and launch the compiled unpacked extension target directly into your chosen Chromium profile layout.</li>
  </ol>

  <h3 style="color: #4A90E2; font-size: 2em; font-weight: 800; margin-top: 30px;">⚙️ Manual Developer Mode Fallback</h3>
  <p style="line-height: 1.8; font-size: 1.05em; color: #333;">If you prefer to load the compiled assets manually into a Chromium framework:</p>
  <ol style="line-height: 2; font-size: 1.05em; color: #333;">
    <li>Navigate to <code style="background-color: #f0f0f0; padding: 3px 8px; border-radius: 4px;">chrome://extensions/</code> in your browser URL bar.</li>
    <li>Enable the <strong>Developer mode</strong> toggle switch in the top-right corner.</li>
    <li>Click the <strong>Load unpacked</strong> button located in the top-left corner.</li>
    <li>Open your extracted workspace folder, navigate to <code style="background-color: #f0f0f0; padding: 3px 8px; border-radius: 4px;">browser_exten/Ui/</code>, and select the newly generated <code style="background-color: #f0f0f0; padding: 3px 8px; border-radius: 4px;">dist</code> directory.</li>
  </ol>
