  <h2 style="color: #333; margin-top: 0;">1. Overview & Philosophy</h2>
  <p style="color: #555; line-height: 1.6;">
    <strong>Ethereal Dashboard</strong> is an ultra-premium, high-performance, privacy-first browser homepage engine designed to execute entirely at the client browser layer. The architecture operates under three strict operational zero-trust principles:
  </p>
  <ul style="color: #555; line-height: 1.6;">
    <li><strong>0 External Server Connections:</strong> All computing overhead and asset loading are sandboxed locally.</li>
    <li><strong>0 Telemetry / Data Tracking:</strong> Guarantees absolute user privacy with no underlying metrics phone-home code.</li>
    <li><strong>0 External API Key Dependencies:</strong> Eliminates operational reliance on paid, rate-limited, or volatile third-party cloud engines.</li>
  </ul>
  <p style="color: #555; line-height: 1.6;">
    <em>Everything executes locally using client compute power, enabling fully deterministic behavior, ultra-low latency canvas rendering, and absolute user data sovereignty.</em>
  </p>
</div>

<br/>

<div style="padding: 20px; border: 1px solid #e1e4e8; border-radius: 8px; margin-bottom: 20px;">
  <h2 style="color: #333; border-bottom: 2px solid #eaecef; padding-bottom: 10px;">2. Technical Capabilities & Core Features</h2>
  
  <h3 style="color: #4A90E2;">Spatial Layout & Canvas Engine</h3>
  <ul style="line-height: 1.6;">
    <li><strong>Premium Glassmorphic UI:</strong> Features fluid, draggable widget orchestration for advanced spatial desktop personalization.</li>
    <li><strong>Fluid Animation Physics:</strong> Utilizes Framer Motion-driven interaction mechanics for natural structural UI transitions.</li>
    <li><strong>GPU-Accelerated Styling:</strong> Implements granular background media filters (blur, contrast, brightness) rendered via hardware-accelerated CSS pipelines to eliminate runtime repaint lag.</li>
  </ul>

  <h3 style="color: #4A90E2;">Client-Side Machine Learning Inference</h3>
  <ul style="line-height: 1.6;">
    <li><strong>Local Foreground Isolation:</strong> Implements fully offline AI subject isolation and background-removal workflows for uploaded custom imagery.</li>
    <li><strong>WASM Neural Network Pipeline:</strong> Executes local WebAssembly neural network inference optimized to remain secure and compliant within strict Chrome Extension Manifest V3 Content Security Policy (CSP) boundaries.</li>
  </ul>

  <h3 style="color: #4A90E2;">Spotlight Search Terminal</h3>
  <ul style="line-height: 1.6;">
    <li><strong>Programmable Slash Commands:</strong> Intercepts and routes custom terminal triggers (such as <code>/yt</code>, <code>/gh</code>, and <code>/g</code>) straight from the interactive inputs.</li>
    <li><strong>Dynamic Asset Ingestion:</strong> Features a processing string injection pipeline that dynamically renders raw local Base64 shortcut icons on active command match detection.</li>
    <li><strong>Safe Dispatch Redirection:</strong> Implements secure query-injection handling for clean, controlled target search engine dispatches.</li>
  </ul>

  <h3 style="color: #4A90E2;">DOM Hijacking Metadata Engine</h3>
  <ul style="line-height: 1.6;">
    <li><strong>Tab Layer Overrides:</strong> Houses a local document-title override engine to bypass standard browser tab text restrictions.</li>
    <li><strong>Canvas-to-Favicon Streaming:</strong> Handles real-time rendering of raw HTML5 canvas Base64 image streams directly into the physical browser tab favicon.</li>
    <li><strong>Zero Network Cost:</strong> Executes visual tab identity updates instantly with zero network roundtrips or static hosting requirements.</li>
  </ul>
</div>

<div style="padding: 20px; border: 1px solid #e1e4e8; border-radius: 8px; margin-bottom: 20px;">
  <h2 style="color: #333;">5. Deployment & Installation Guide</h2>

  <h3 style="color: #4A90E2;">📂 Initial Project Download</h3>
  <p style="line-height: 1.6;">Before proceeding to browser installation, pull the portable project files onto your local machine:</p>
  <ol style="line-height: 1.6;">
    <li>Click the green <strong>Code</strong> button at the top right of this repository.</li>
    <li>Select <strong>Download ZIP</strong> from the dropdown menu.</li>
    <li>Extract the downloaded <code>.zip</code> file archive to your preferred local storage directory.</li>
  </ol>

  <h3 style="color: #4A90E2;">🛠️ Automated Setup <em>(Chromium Browsers: Chrome, Brave, Edge, Opera)</em></h3>
  <p style="line-height: 1.6;">The repository contains an automated orchestration pipeline designed to handle system environment setup and asset bundling automatically.</p>
  <ol style="line-height: 1.6;">
    <li>Open your extracted project directory and navigate into the root <code>browser_exten</code> folder.</li>
    <li>Locate the file named <code>install_and_launch.bat</code>.</li>
    <li>Right-click on the script and select <strong>"Run as Administrator"</strong>.</li>
    <li>The script will automatically verify dependencies, parse project files, and execute a production compile using the following routines:
<pre style="background-color: #2b2b2b; color: #f8f8f2; padding: 10px; border-radius: 5px;"><code>pnpm install
pnpm build</code></pre>
    </li>
    <li>On completion, it will natively inject and launch the compiled unpacked extension target directly into your chosen Chromium profile layout.</li>
  </ol>

  <h3 style="color: #4A90E2;">⚙️ Manual Developer Mode Fallback</h3>
  <p style="line-height: 1.6;">If you prefer to load the compiled assets manually into a Chromium framework:</p>
  <ol style="line-height: 1.6;">
    <li>Navigate to <code>chrome://extensions/</code> in your browser URL bar.</li>
    <li>Enable the <strong>Developer mode</strong> toggle switch in the top-right corner.</li>
    <li>Click the <strong>Load unpacked</strong> button located in the top-left corner.</li>
    <li>Open your extracted workspace folder, navigate to <code>browser_exten/Ui/</code>, and select the newly generated <code>dist</code> directory.</li>
  </ol>

  <h3 style="color: #e34f26;">🦊 Manual Warning-Free Setup <em>(Mozilla Firefox)</em></h3>
  <p style="line-height: 1.6; background-color: #fff3cd; border-left: 4px solid #ffeeba; padding: 10px; color: #856404;">
    <strong>Warning:</strong> Automated command-line injection tools trigger external developer debugging hooks within Gecko architectures, forcing a permanent and visually disruptive "Browser is under remote control" banner across your viewport.
  </p>
  <p style="line-height: 1.6;">For a completely clean, pristine, and warning-free installation inside Firefox, execute these manual activation steps:</p>
  <ol style="line-height: 1.6;">
    <li>Open Mozilla Firefox normally from your desktop or taskbar shortcut.</li>
    <li>In the URL address bar at the top, type exactly <code>about:debugging</code> and press Enter.</li>
    <li>On the left-hand navigation menu pane, click on <strong>This Firefox</strong>.</li>
    <li>Find the <strong>Temporary Extensions</strong> header block and click the <strong>Load Temporary Add-on...</strong> button.</li>
    <li>An OS file explorer window will open. Navigate directly into the compiled production asset directory where you extracted your project zip file:<br/>
      <code style="background-color: #f6f8fa; padding: 2px 5px; border-radius: 3px;">...\browser_exten\Ui\dist\</code>
    </li>
    <li>Locate and double-click the <code>manifest.json</code> file inside that <code>Ui/dist/</code> directory.</li>
    <li>Open a new tab (<kbd>Ctrl</kbd> + <kbd>T</kbd>) to immediately experience your premium, edge-to-edge dashboard layout.</li>
  </ol>
